The Free Software Foundation today announced their evaluations of major code repository-hosting services per the standards of the GNU Ethical Criteria for Code Repositories. Ranked as part of the code repository platforms were GNU Savannah, GitLab, GitHub, and SourceForge. The only code hosting platform to receive an A rating for "ethical hosting" was their own GNU Savannah platform.
